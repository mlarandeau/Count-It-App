Recordings

Count It allows you to record the frame rate of a game in order to view and analyze later. Recoding a frame rate is separate than tracking a frame rate and must be explicitly started. Tracking the frame rate does automatically start a recording.

To start recording, start tracking the frame rate and then either use the record button in the FPS display or use the hotkey that you specify in "Preferences".

For more details on starting a recording, see Recording Preferences.
