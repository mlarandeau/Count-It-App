Exporting a Recording


Recordings can be exported in a few different formats that makes it easy to archive and share frame rate data.

To access the export options, select a recording then choose "Recording" > "Export" from the main menu and choose the format you would like the data exported as. Alternatively, right-click a recording and select "Export" and then choose the desired file format.

as csv:
The recording data is exported as a csv (comma separated) file that can be easily opened in any spreadsheet application. The frame rate is recorded in time intervals, set by the "Sampling Rate" option in the General tab of the Preferences window. The sample weight column specifies the number of frame used to calculate the frame rate for that sample. The absolute maximum, absolute minimum, and absolute average values are based on measuring every single frame. The sample maximum, sample minimum, and sample average are based on the sample frame rates.


Note: The frame rate of individual frames is not recorded in order to maintain good performance over extended recording sessions.


as png/jpeg:
When exported as an image file type, the image is based on the current zoom settings set by the "Zoom" toolbar drop down. The computer details are included in the exported images.
