Recording Preferences


Start/Stop Recording Sessions:
  • Auto start recording:
    Tracking the frame rate of a game will also starts a recording session.
  • Enable in game recording with Hotkey
    Allows you to use a hotkey to start and stop recording sessions while in a game. The first time this option is selected, you will be prompted to enable Count It as an accessibility app. You must enable it in order for the hotkey option to function.

    The hotkey can be customized but must be a single character. The hotkey is always prefixed with opt-cmd.

  • Display Recording Button in FPS Display:
    A button will be placed in the frame rate counter display. Clicking this button allows you to start and stop a recording session while in a game.


Record Button:
By default, a button is placed in the frame rate display that can be used to start and stop a recording session while tracking a game's frame rate.
  • Primary Color:
    Sets the color of the button. This will also be used as the background color of the FPS display when "Use for FPS display while recording" is enabled.

  • Secondary Color:
    Sets the border color of the button. This will also be used as the FPS text color when "Use for FPS display while recording" is enabled.

  • Use for FPS display while recording:
    When enabled, these same colors are used to alter the color of the FPS display. Changing the color of the FPS display while recording can be useful to make it more obvious that a recording is in process, especially if "Display Recording Button in FPS Display" is disabled.

Note: Depending on how a particular game uses the mouse, this button may be inaccessible. In that case, use the hotkey option above.

Note: Transparency (opacity) can be adjusted using the "opacity" slider when setting the color of the background or the text.


Auto Export Recordings to:
When selected, a csv report of the recording will automatically be created when the recording ends. A subfolder is created for each game when a report is created
