FPS Display Preferences


Screen Position:
Sets the position that the frame rate is displayed on the screen. This position is relative to the monitor, not the game window.

Style:
  • Background Color:
    Sets the color of the background that is drawn around the frame rate.

  • Rounded Corners:
    Sets whether or not the background should be drawn with rounded (checked) or sharp (unchecked) corners.

  • Text Color:
    Sets the color of the frame rate.

  • Text Size:
    Set the size of the frame rate text.

  • Font:
    Set the font used for displaying the frame rate count.
Note: Transparency (opacity) can be adjusted using the "opacity" slider when setting the color of the background or the text.
