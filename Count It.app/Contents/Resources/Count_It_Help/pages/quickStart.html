Quick Start
  1. Start the game that you want to track the frame rate of (the game may need to be in windowed mode this first time)
  2. Launch Count It
  3. Click the "+" icon in the lower left corner of the main window
  4. Choose "Running..."
  5. Find the game from the list and click "Add"
  6. Select the game from the main window and click the "Start" button in the toolbar.
  7. A spinner icon will appear while Count It connects to the game.
  8. Once connected, the frame rate will display in the lower left corner (default) of your screen.


Note: If you are having trouble finding the game to add, go to Adding a Game to view more options.
