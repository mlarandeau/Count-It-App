Adding a Game


In order to track the frame rate of a game, it must first be added to Count It. A game only has to be added once.

From the main menu select "Game" > "Add Game"
(Alternatively, click the "+" icon in the lower left corner of the main window.)


Browse:
If you know where the game is located on your computer, you can use the "Browse..." to find the game in the finder and click "Add".


Platform:
Use this option to add a game that is managed using a third-party platform.
(Currently supported platforms: Steam, MacGameStore, GOG Galaxy, Battle.net)

After clicking "Platform...", you will be asked to select from a list of the supported platforms that were found on your computer.

Choose a platform and click "Next"
Choose a game(s) from the list of games that were found on your computer from that platform, and click "Add"


Running:
This option displays all currently running applications. This can make it easier to find the game if the game is already running.

After clicking "Running...", a list of all currently running applications will be displayed. Find the game you are looking for and click "Add".


Drag and Drop:
A game(s) can be added by clicking and dragging the game from Finder into the games list of the main window.


Note: If you cannot find the game you are looking for when using the "Running" or "Platform" option, click "Show hidden apps" while viewing the list of games found for that service. This may be necessary with some games that display a launcher application before the game loads. Also, only games that are bundled as apps (have the .app extension) can be tracked by Count It. Games that are compiled as executable files are not supported.

If you cannot find the game using any of the above options:
  1. Launch the game that you want to track
  2. cmd-click the game's icon in the dock (the game may have to be in windowed mode in order to get to the dock). This will show the game in Finder.
  3. Drag the game from Finder into the Count It
