Tracking the Frame Rate


In order to track the frame rate, the game must first be added to Count It. Go to Adding a Game to learn how to add a game.

Select a game and then use the toolbar buttons to start tracking the frame rate. By default, the frame rate is displayed in the lower left corner of your screen. The location and other options can be modified using the preference menu.

Only one game can be tracked at a time.


Launch:
If the game is not yet running, clicking "Launch" will first launch the game. Once the game is launched, Count It will start tracking the frame rate.

Note: If you use a third-party platform (i.e. Steam) to play the game, Count It should not be used to launch the game. See "Wait" below.


Start:
If the game is already running, click "Start" to begin tracking the frame rate.


Stop:
Stop tracking the frame rate at any time by clicking "Stop"


Wait:
Choosing "Wait" tells Count It to wait for the selected game to launch. Once the game launches, Count It will begin tracking the frame rate.

This is most useful in two situations:
  1. If you use a third-party platform to play your games, you will not be able to take advantage the platform's' features if you launch the game from Count It. Instead, select the game, choose "Wait", and then launch the game using the launch feature of the platform.
  2. Many games do not allow you to switch between apps when playing in fullscreen mode. Using the "Wait" option allows you to launch a game directly into fullscreen mode without having to switch back to Count It to start tracking the frame rate.
